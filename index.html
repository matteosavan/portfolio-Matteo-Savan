Projeto de Machine Learning — Classificação de Estados Fisiológicos

Desenvolvi um modelo de aprendizado de máquina para classificação de estados fisiológicos (estresse, atividade aeróbica e anaeróbica) a partir de dados reais e ruidosos coletados por smartwatch usando Python (NumPy, Pandas, scickit-learn). O projeto exigiu limpeza, análise exploratória dos dados, seleção de atributos relevantes, normalização, avaliação de múltiplos modelos e análise de underfitting e overfitting.
