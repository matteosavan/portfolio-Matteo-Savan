## Estrutura

- `index.html`: pagina principal.
- `personalization.css`: estilos do site.
- `script.js`: scripts e interacoes.
- `images/`: imagens usadas no portfólio.

## Como abrir

1. Baixe ou clone o repositorio.
2. Abra o arquivo `index.html` no navegador.

## Personalizacao

- Edite textos e links no `index.html`.
- Ajuste cores, fontes e layout em `personalization.css`.
- Adicione novas imagens em `images/` e referencie no HTML.
